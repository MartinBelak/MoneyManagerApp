# Money Manager App
In this little side project I trained my JavaScript skills. Using free Bootstrap template that can be found [here.](https://bootstrapmade.com/demo/Techie/)
I created web application that can help you manage your finances

## Tutorial

After opening index.html

Navigate to Overview where you can create new month by typing a name and income for that month/time period.

You can then select that month to add categories and expenses

Total of spendings/savings is displayed on the Overview page
